## 2. 安装依赖

### 2.1 创建虚拟环境（推荐）

```bash
# 方法 1：使用 venv
python -m venv venv

# 激活虚拟环境
# Linux/Mac:
source venv/bin/activate

# Windows:
venv\Scripts\activate

# 方法 2：使用 conda
conda create -n a-stock-quant python=3.10
conda activate a-stock-quant
```

### 2.2 安装 Python 依赖

```bash
# 升级 pip
pip install --upgrade pip

# 安装核心依赖
pip install -r requirements.txt

# 安装开发依赖（可选）
pip install -r requirements-dev.txt
```

### 2.3 依赖说明

| 依赖包 | 用途 | 版本 |
|--------|------|------|
| pandas | 数据处理 | >=2.0 |
| numpy | 数值计算 | >=1.24 |

| tushare | 备用数据源 | >=1.2 |

## 3. 配置数据源

### 3.1 AKShare（推荐，免费）

AKShare 是免费开源的数据接口，无需配置即可使用：

```python
import akshare as ak

# 测试获取数据
stock_info = ak.stock_info_a_code_name()
print(stock_info.head())
```

### 3.2 Tushare Pro（可选，需要积分）

Tushare Pro 提供更稳定的数据服务，但需要注册获取 token：

1. 访问 https://tushare.pro/ 注册账号
2. 获取个人 token
3. 配置 token：

```bash
# 创建配置文件
cp config/tushare.yaml.example config/tushare.yaml

# 编辑配置文件，填入你的 token
vim config/tushare.yaml
```

```yaml
# config/tushare.yaml
token: "your_tushare_token_here"
```

### 3.3 数据缓存配置

```yaml
# config/data_cache.yaml
cache:
  enabled: true
  cache_dir: "data/cache"
  expire_hours: 24  # 缓存过期时间（小时）
```

### 4.4 修改策略参数

在代码中调整策略参数：

```python
from strategies.trend_following.trend_strategy import TrendFollowingStrategy, TrendFollowingConfig

# 创建自定义配置
config = TrendFollowingConfig(
    ma_short=10,          # 短期均线
    ma_long=30,           # 长期均线
    stop_loss_ratio=0.05, # 止损比例（5%）
    take_profit_ratio=0.15 # 止盈比例（15%）
)

# 创建策略实例
strategy = TrendFollowingStrategy(config)
```

### 5.3 常见问题

**Q: 数据获取失败怎么办？**

A: 
1. 检查网络连接
2. 检查 AKShare 版本是否最新
3. 尝试使用缓存数据

**Q: 回测结果不理想怎么办？**

A:
1. 检查策略参数是否合理
2. 查看交易记录分析问题
3. 调整策略逻辑

**Q: 如何添加新策略？**

A:
1. 在 `strategies/` 目录创建新文件夹
2. 参考现有策略结构创建文件
3. 实现 `generate_signals()` 方法
4. 编写回测脚本
