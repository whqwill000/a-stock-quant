## 模拟器配置

### 1. 初始资金配置

```python
from core.simulator.account import Account

# 创建账户
account = Account(
    initial_cash=1000000  # 初始资金 100 万元
)

# 查看账户信息
print(f"初始资金: {account.initial_cash:,.2f} 元")
print(f"可用资金: {account.available_cash:,.2f} 元")
print(f"总市值: {account.total_asset:,.2f} 元")
```

**资金配置建议**：

| 资金规模 | 适用场景 | 风险等级 |
|---------|---------|---------|
| 10 万元 | 新手练习 | 低 |
| 50 万元 | 进阶用户 | 中 |
| 100 万元 | 专业交易 | 中高 |
| 500 万元 | 大资金操作 | 高 |

### 2. 交易成本配置

A 股交易成本包括：

- **佣金**：万 2.5（最低 5 元）
- **印花税**：0.05%
- **过户费**：十万分之一

```python
from core.simulator.account import Account

account = Account(
    initial_cash=1000000,
    commission_rate=0.00025,    # 佣金：万 2.5
    min_commission=5.0,          # 最低佣金：5 元
    stamp_tax_rate=0.0005,       # 印花税：0.05%
    transfer_fee_rate=0.00001    # 过户费：十万分之一
)
```

### 3. 风控配置

```python
from core.simulator.risk_control import RiskControl, RiskConfig

# 创建风控配置
risk_config = RiskConfig(
    max_position_ratio=0.30,    # 单只股票最大仓位 30%
    max_total_position=0.95,    # 最大总仓位 95%
    max_loss_ratio=0.10,        # 最大亏损比例 10%
    stop_loss_threshold=0.05,   # 止损阈值 5%
    take_profit_threshold=0.15  # 止盈阈值 15%
)

# 创建风控模块
risk_control = RiskControl(risk_config)
```

**风控配置建议**：

| 风险偏好 | 单股仓位 | 总仓位 | 止损阈值 |
|---------|---------|--------|---------|
| 保守型 | 10% | 50% | 3% |
| 稳健型 | 20% | 70% | 5% |
| 平衡型 | 30% | 80% | 8% |
| 激进型 | 50% | 95% | 10% |

## 策略选择

### 趋势跟踪策略

**适用场景**：明显的上涨或下跌趋势行情

**策略特点**：
- 买入后持有，顺势而为
- 适合牛市和熊市
- 震荡市表现较差

**配置示例**：

```python
from strategies import TrendFollowingStrategy

strategy = TrendFollowingStrategy(
    ma_short=5,      # 短期均线
    ma_long=20,      # 长期均线
    entry_threshold=0.02,  # 入场阈值
    exit_threshold=0.05    # 出场阈值
)
```

**推荐配置**：

| 市场环境 | ma_short | ma_long | 仓位 |
|---------|---------|---------|------|
| 牛市 | 5 | 20 | 50-80% |
| 熊市 | 5 | 20 | 20-40% |
| 震荡市 | 10 | 30 | 10-30% |

### 均值回归策略

**适用场景**：震荡行情、均值回归明显的股票

**策略特点**：
- 高抛低吸
- 适合震荡市
- 需要良好的择时能力

**配置示例**：

```python
from strategies import MeanReversionStrategy

strategy = MeanReversionStrategy(
    ma_period=20,           # 均线周期
    entry_zscore=2.0,       # 入场 Z-score
    exit_zscore=0.5,        # 出场 Z-score
    max_position=0.3        # 最大仓位
)
```

**推荐配置**：

| 市场环境 | ma_period | entry_zscore | 仓位 |
|---------|-----------|--------------|------|
| 强势震荡 | 10 | 1.5 | 30-50% |
| 弱势震荡 | 20 | 2.0 | 20-40% |
| 波动剧烈 | 30 | 2.5 | 10-30% |

### 多因子策略

**适用场景**：专业投资者、机构用户

**策略特点**：
- 综合考虑多个因子
- 适合中长期投资
- 需要较多数据支持

**配置示例**：

```python
from strategies import MultiFactorStrategy

strategy = MultiFactorStrategy(
    factors=['value', 'momentum', 'quality', 'volatility'],
    weights=[0.3, 0.25, 0.25, 0.2],
    top_n=10,               # 选股数量
    rebalance_freq=20       # 再平衡频率（交易日）
)
```

**推荐配置**：

| 投资风格 | 因子权重 | 选股数量 | 再平衡频率 |
|---------|---------|---------|-----------|
| 价值投资 | value:0.4 | 20 | 30 |
| 成长投资 | momentum:0.4 | 15 | 20 |
| 均衡配置 | 多因子均衡 | 10 | 10 |

### 动量策略

**适用场景**：强势行情、热点轮动

**策略特点**：
- 追涨杀跌
- 适合热点轮动
- 需要快速反应

**配置示例**：

```python
from strategies import MomentumStrategy

strategy = MomentumStrategy(
    lookback_period=60,     # 回看周期
    top_n=5,                # 选股数量
    holding_period=20,      # 持仓周期
    momentum_threshold=0.1  # 动量阈值
)
```

**推荐配置**：

| 市场环境 | lookback_period | holding_period | 仓位 |
|---------|-----------------|----------------|------|
| 强势行情 | 30 | 10 | 50-80% |
| 一般行情 | 60 | 20 | 30-50% |
| 弱势行情 | 90 | 30 | 10-30% |

### 套利策略

**适用场景**：专业投资者、机构用户

**策略特点**：
- 低风险低收益
- 需要快速执行
- 适合大资金

**配置示例**：

```python
from strategies import StatisticalArbitrageStrategy

strategy = StatisticalArbitrageStrategy(
    pair_threshold=2.0,     # 价差阈值
    entry_threshold=1.5,    # 入场阈值
    exit_threshold=0.5,     # 出场阈值
    max_position=0.2        # 最大仓位
)
```

### 事件驱动策略

**适用场景**：重大事件、政策变化

**策略特点**：
- 抓住事件机会
- 高风险高收益
- 需要快速决策

**配置示例**：

```python
from strategies import EventDrivenStrategy

strategy = EventDrivenStrategy(
    event_types=['earnings', 'policy', 'merger'],
    holding_period=10,      # 持仓周期
    position_ratio=0.3,     # 仓位比例
    stop_loss=0.08          # 止损阈值
)
```

### 资金流策略

**适用场景**：追踪主力资金动向

**策略特点**：
- 跟随主力资金
- 中短期操作
- 需要实时数据

**配置示例**：

```python
from strategies import CapitalFlowStrategy

strategy = CapitalFlowStrategy(
    main_force_threshold=0.05,  # 主力资金阈值
    holding_period=15,          # 持仓周期
    position_ratio=0.3,         # 仓位比例
    stop_loss=0.06              # 止损阈值
)
```

## 运行回测

### 使用 Python 脚本

```python
from core.backtest.engine import BacktestEngine, BacktestConfig
from strategies import TrendFollowingStrategy

# 1. 配置回测
config = BacktestConfig(
    initial_cash=1000000,
    start_date="2023-01-01",
    end_date="2023-12-31",
    commission_rate=0.00025,
    stamp_tax_rate=0.0005
)

# 2. 创建引擎
engine = BacktestEngine(config)

# 3. 加载数据
engine.load_data(your_data)

# 4. 创建策略
strategy = TrendFollowingStrategy()

# 5. 运行回测
result = engine.run_backtest(strategy)

# 6. 查看结果
print(result.summary())
```

### 使用 Jupyter Notebook

```python
# 在 Jupyter Notebook 中
%matplotlib inline

from core.backtest.engine import BacktestEngine, BacktestConfig
from strategies import MeanReversionStrategy
import pandas as pd

# 加载数据
data = pd.read_csv('your_data.csv')

# 配置回测
config = BacktestConfig(
    initial_cash=500000,
    start_date="2023-01-01",
    end_date="2023-12-31"
)

# 创建引擎
engine = BacktestEngine(config)
engine.load_data(data)

# 创建策略
strategy = MeanReversionStrategy()

# 运行回测
result = engine.run_backtest(strategy)

# 可视化结果
result.plot_equity_curve()
result.plot_drawdown()
result.plot_returns_distribution()
```

## 常见问题

### Q1: 如何选择初始资金？

**A**：建议根据以下原则选择：

1. **新手**：10-50 万元，练习为主
2. **进阶**：50-100 万元，验证策略
3. **专业**：100 万元以上，实盘前测试

### Q2: 如何调整仓位？

**A**：根据市场环境和策略特点调整：

```python
# 牛市增加仓位
if market_trend == "bull":
    position_ratio = 0.7
# 熊市减少仓位
elif market_trend == "bear":
    position_ratio = 0.3
# 震荡市中等仓位
else:
    position_ratio = 0.5
```

### Q3: 如何设置止损？

**A**：根据策略特点设置：

```python
# 趋势策略：较宽松止损
stop_loss = 0.08  # 8%

# 均值回归：较严格止损
stop_loss = 0.05  # 5%

# 套利策略：极严格止损
stop_loss = 0.03  # 3%
```

### Q4: 如何评估策略有效性？

**A**：使用以下标准：

1. **夏普比率 > 1.0**：风险调整收益良好
2. **最大回撤 < 20%**：风险可控
3. **胜率 > 50%**：策略稳定
4. **盈亏比 > 1.5**：盈利空间充足

### Q5: 如何优化策略参数？

**A**：使用网格搜索：

```python
from core.backtest.engine import GridSearch

# 定义参数范围
param_grid = {
    'ma_short': [5, 10, 20],
    'ma_long': [20, 30, 50]
}

# 网格搜索
search = GridSearch(TrendFollowingStrategy, param_grid)
best_params = search.search(data)
```
