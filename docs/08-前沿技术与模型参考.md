### 1.5 AKQuant (高性能量化回测框架) ⭐⭐⭐⭐⭐

**GitHub**: https://github.com/akfamily/AKQuant

**特点**:
- **Rust核心引擎** - 零开销抽象，回测速度大幅提升
- **Zero-Copy架构** - 减少内存拷贝，提升性能
- **Polars驱动** - 高性能因子计算引擎
- **原生ML支持** - 内置Walk-forward Validation框架
- **因子表达式引擎** - 支持Alpha101风格公式
- **专业级风控** - 订单流管理与即时风控模块
- **可视化模块** - 基于Plotly的交互式HTML报告

**安装**:
```bash
pip install akquant
```

**使用示例**:
```python
import akquant as aq
import akshare as ak

# 准备数据
df = ak.stock_zh_a_daily(symbol="sh600000", start_date="20250212", end_date="20260212")

class MyStrategy(aq.Strategy):
    def on_bar(self, bar):
        current_pos = self.get_position(bar.symbol)
        if current_pos == 0 and bar.close > bar.open:
            self.buy(symbol=bar.symbol, quantity=100)
        elif current_pos > 0 and bar.close < bar.open:
            self.close_position(symbol=bar.symbol)

# 运行回测
result = aq.run_backtest(
    data=df,
    strategy=MyStrategy,
    initial_cash=100000.0,
    symbol="sh600000"
)
```

### 3.2 FinBERT (金融情感分析) ⭐⭐⭐⭐⭐

**HuggingFace**: https://huggingface.co/prosusai/finbert

**GitHub**: https://github.com/prosusai/finbert

**特点**:
- 金融领域专用 BERT
- 情感分析（正面/负面/中性）
- 基于 Reuters TRC2 数据集训练

**安装与使用**:
```python
from transformers import AutoTokenizer, AutoModelForSequenceClassification
import torch

# 加载模型
tokenizer = AutoTokenizer.from_pretrained("ProsusAI/finbert")
model = AutoModelForSequenceClassification.from_pretrained("ProsusAI/finbert")

# 情感分析
text = "Tesla's stock surged after strong earnings report."
inputs = tokenizer(text, return_tensors="pt", truncation=True, padding=True)

with torch.no_grad():
    outputs = model(**inputs)
    probs = torch.softmax(outputs.logits, dim=-1)
    print(f"Positive: {probs[0][2].item():.3f}")
    print(f"Negative: {probs[0][0].item():.3f}")
    print(f"Neutral: {probs[0][1].item():.3f}")
```

## 10. 本项目推荐技术栈

基于前沿调研，本项目推荐以下技术栈：

### 核心框架
- **数据处理**: pandas + numpy + polars (高性能)
- **数据源**: AKShare (免费) + Tushare Pro (备用) + AKQuant (高性能回测)
- **回测引擎**: AKQuant (Rust内核) / 自研 (参考 Qlib 设计)

### 机器学习
- **树模型**: LightGBM (多因子选股)
- **深度学习**: PyTorch + LSTM/Transformer/TimeXer (时间序列)
- **强化学习**: Stable-Baselines3 / FinRL-Meta (策略优化)
- **LLM应用**: RD-Agent-Quant (自动化因子挖掘)

### NLP
- **情感分析**: FinBERT (ProsusAI) + CHINESE-CRYPTO-SENTIMENT
- **文本分类**: Qwen3-8B + rLoRA + FINANCE-LLAMA3-8B
- **自动化因子挖掘**: RD-Agent-Quant

### 可视化
- **绩效分析**: QuantStats
- **图表**: matplotlib + plotly (交互式)

### 时间序列模型
- **基础模型**: PatchTST, TimesNet, iTransformer
- **最新模型**: TimeMixer, TimeXer, Chronos2, TiRex

## 12. 注意事项

⚠️ **重要提示**:

1. **数据质量**: 免费数据可能存在质量问题，建议使用前进行数据清洗
2. **过拟合风险**: 机器学习模型容易过拟合，需使用交叉验证
3. **交易成本**: 回测时需考虑佣金、印花税、滑点等成本
4. **实盘谨慎**: 回测结果不代表实盘表现，实盘前需充分模拟
5. **模型时效**: 市场风格会变化，模型需定期更新
