### 1.1 Qlib (微软 AI 量化投资平台) ⭐⭐⭐⭐⭐

**GitHub**: https://github.com/microsoft/qlib (38.2k+ stars)

**特点**:
- AI 导向的量化投资平台
- 支持监督学习、强化学习、市场动态建模
- 提供完整的数据处理、模型训练、回测流程
- 支持 A 股和中国市场

**安装**:
```bash
# pip 安装
pip install pyqlib

# 源码安装（推荐）
git clone https://github.com/microsoft/qlib.git && cd qlib
pip install -e .[dev]
```

**获取 A 股数据**:
```bash
# 获取日线数据
python -m qlib.cli.data qlib_data \
  --target_dir ~/.qlib/qlib_data/cn_data \
  --region cn

# 获取 1 分钟高频数据
python -m qlib.cli.data qlib_data \
  --target_dir ~/.qlib/qlib_data/cn_data_1min \
  --region cn --interval 1min
```

**内置模型**:
- 树模型：XGBoost, LightGBM, Catboost, DoubleEnsemble
- 深度学习：MLP, LSTM, GRU, ALSTM, TCN, Transformer, TFT, TabNet
- 图神经网络：GATs, SFM
- 市场动态：TCTS, TRA, ADD, DDG-DA, IGMTF, HIST

**文档**: https://qlib.readthedocs.io/

### 3.2 FinBERT (金融情感分析) ⭐⭐⭐⭐⭐

**HuggingFace**: https://huggingface.co/prosusai/finbert

**GitHub**: https://github.com/prosusai/finbert

**特点**:
- 金融领域专用 BERT
- 情感分析（正面/负面/中性）
- 基于 Reuters TRC2 数据集训练

**安装与使用**:
```python
from transformers import AutoTokenizer, AutoModelForSequenceClassification
import torch

# 加载模型
tokenizer = AutoTokenizer.from_pretrained("ProsusAI/finbert")
model = AutoModelForSequenceClassification.from_pretrained("ProsusAI/finbert")

# 情感分析
text = "Tesla's stock surged after strong earnings report."
inputs = tokenizer(text, return_tensors="pt", truncation=True, padding=True)

with torch.no_grad():
    outputs = model(**inputs)
    probs = torch.softmax(outputs.logits, dim=-1)
    print(f"Positive: {probs[0][2].item():.3f}")
    print(f"Negative: {probs[0][0].item():.3f}")
    print(f"Neutral: {probs[0][1].item():.3f}")
```

## 12. 注意事项

⚠️ **重要提示**:

1. **数据质量**: 免费数据可能存在质量问题，建议使用前进行数据清洗
2. **过拟合风险**: 机器学习模型容易过拟合，需使用交叉验证
3. **交易成本**: 回测时需考虑佣金、印花税、滑点等成本
4. **实盘谨慎**: 回测结果不代表实盘表现，实盘前需充分模拟
5. **模型时效**: 市场风格会变化，模型需定期更新
