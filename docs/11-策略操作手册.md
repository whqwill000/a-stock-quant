## 策略选择指南

### 根据市场环境选择

```python
def select_strategy_by_market(market_condition):
    """根据市场条件选择策略"""
    if market_condition == "bull":
        return "trend_following"  # 牛市：趋势跟踪
    elif market_condition == "bear":
        return "trend_following"  # 熊市：趋势跟踪（做空）
    elif market_condition == "ranging":
        return "mean_reversion"   # 震荡：均值回归
    elif market_condition == "volatile":
        return "momentum"         # 波动：动量策略
    else:
        return "multi_factor"     # 混合：多因子
```

### 根据风险偏好选择

```python
def select_strategy_by_risk(risk_level):
    """根据风险偏好选择策略"""
    if risk_level == "conservative":
        return "mean_reversion"   # 保守型：均值回归
    elif risk_level == "moderate":
        return "trend_following"  # 稳健型：趋势跟踪
    elif risk_level == "aggressive":
        return "momentum"         # 激进型：动量策略
    else:
        return "multi_factor"     # 平衡型：多因子
```

### 根据资金规模选择

```python
def select_strategy_by_capital(capital):
    """根据资金规模选择策略"""
    if capital < 500000:
        return "mean_reversion"   # 小资金：均值回归
    elif capital < 2000000:
        return "trend_following"  # 中等资金：趋势跟踪
    else:
        return "multi_factor"     # 大资金：多因子
```

## 策略详细说明

### 1. 趋势跟踪策略

**策略原理**：
- 基于均线、通道等趋势指标
- 价格突破均线或通道时入场
- 趋势反转时出场
- 配合止损止盈控制风险

**适用场景**：
- 明显的上涨或下跌趋势行情
- 牛市和熊市
- 震荡市表现较差

**核心参数**：

| 参数 | 说明 | 默认值 | 调整建议 |
|-----|------|-------|---------|
| ma_short | 短期均线周期 | 5 | 牛市减小，熊市增大 |
| ma_long | 长期均线周期 | 20 | 牛市减小，熊市增大 |
| entry_threshold | 入场阈值 | 0.02 | 波动大时增大 |
| exit_threshold | 出场阈值 | 0.05 | 严格止损时减小 |
| stop_loss | 止损阈值 | 0.08 | 保守型减小 |
| take_profit | 止盈阈值 | 0.15 | 激进型增大 |

**配置示例**：

```python
from strategies import TrendFollowingStrategy

# 牛市配置
bull_config = {
    'ma_short': 5,
    'ma_long': 20,
    'entry_threshold': 0.02,
    'exit_threshold': 0.05,
    'stop_loss': 0.08,
    'take_profit': 0.20
}

# 熊市配置
bear_config = {
    'ma_short': 5,
    'ma_long': 20,
    'entry_threshold': 0.02,
    'exit_threshold': 0.05,
    'stop_loss': 0.05,
    'take_profit': 0.10
}

# 创建策略
strategy = TrendFollowingStrategy(**bull_config)
```

**操作步骤**：

1. **初始化策略**
   ```python
   strategy = TrendFollowingStrategy(ma_short=5, ma_long=20)
   ```

2. **运行回测**
   ```python
   result = engine.run_backtest(strategy)
   ```

3. **查看信号**
   ```python
   signals = strategy.generate_signals(data)
   for signal in signals:
       print(f"{signal.stock_code}: {signal.signal_type} @ {signal.price}")
   ```

### 2. 均值回归策略

**策略原理**：
- 基于统计学均值回归理论
- 价格偏离均值时入场
- 回归均值时出场
- 常用指标：RSI、布林带、Z-score

**适用场景**：
- 震荡行情
- 均值回归明显的股票
- 波动率适中的市场

**核心参数**：

| 参数 | 说明 | 默认值 | 调整建议 |
|-----|------|-------|---------|
| ma_period | 均线周期 | 20 | 波动大时增大 |
| entry_zscore | 入场 Z-score | 2.0 | 波动大时增大 |
| exit_zscore | 出场 Z-score | 0.5 | 严格出场时减小 |
| rsi_period | RSI 周期 | 14 | 常用 14 |
| oversold | 超卖阈值 | 30 | 保守型减小 |
| overbought | 超买阈值 | 70 | 激进型增大 |
| bollinger_std | 布林带标准差 | 2.0 | 波动大时增大 |

**配置示例**：

```python
from strategies import MeanReversionStrategy

# 震荡市配置
ranging_config = {
    'ma_period': 20,
    'entry_zscore': 2.0,
    'exit_zscore': 0.5,
    'rsi_period': 14,
    'oversold': 30.0,
    'overbought': 70.0
}

# 创建策略
strategy = MeanReversionStrategy(**ranging_config)
```

**操作步骤**：

1. **初始化策略**
   ```python
   strategy = MeanReversionStrategy(ma_period=20, entry_zscore=2.0)
   ```

2. **运行回测**
   ```python
   result = engine.run_backtest(strategy)
   ```

3. **查看信号**
   ```python
   signals = strategy.generate_signals(data)
   ```

### 3. 多因子策略

**策略原理**：
- 综合考虑多个因子
- 基于因子打分选股
- 定期再平衡组合
- 适合中长期投资

**适用场景**：
- 专业投资者
- 机构用户
- 中长期投资

**核心因子**：

| 因子 | 说明 | 适用场景 |
|-----|------|---------|
| value | 价值因子 | 价值投资 |
| momentum | 动量因子 | 成长投资 |
| quality | 质量因子 | 稳健投资 |
| volatility | 波动因子 | 风险控制 |

**核心参数**：

| 参数 | 说明 | 默认值 | 调整建议 |
|-----|------|-------|---------|
| factors | 因子列表 | ['value', 'momentum', 'quality', 'volatility'] | 根据投资风格调整 |
| weights | 因子权重 | [0.3, 0.25, 0.25, 0.2] | 根据因子重要性调整 |
| top_n | 选股数量 | 10 | 小资金减少，大资金增加 |
| rebalance_freq | 再平衡频率 | 20 | 高频交易减小 |
| min_market_cap | 最小市值 | 1000000000 | 小盘股增加 |

**配置示例**：

```python
from strategies import MultiFactorStrategy

# 价值投资配置
value_investing = {
    'factors': ['value', 'quality', 'volatility'],
    'weights': [0.5, 0.3, 0.2],
    'top_n': 20,
    'rebalance_freq': 30
}

# 成长投资配置
growth_investing = {
    'factors': ['momentum', 'quality', 'value'],
    'weights': [0.4, 0.35, 0.25],
    'top_n': 15,
    'rebalance_freq': 20
}

# 创建策略
strategy = MultiFactorStrategy(**value_investing)
```

**操作步骤**：

1. **初始化策略**
   ```python
   strategy = MultiFactorStrategy(
       factors=['value', 'momentum', 'quality', 'volatility'],
       top_n=10,
       rebalance_freq=20
   )
   ```

2. **运行回测**
   ```python
   result = engine.run_backtest(strategy)
   ```

3. **查看选股**
   ```python
   stocks = strategy.select_stocks(data)
   print(f"选中股票: {stocks}")
   ```

### 5. 套利策略

**策略原理**：
- 利用价格差异套利
- 低风险低收益
- 需要快速执行
- 适合大资金

**适用场景**：
- 专业投资者
- 机构用户
- 大资金操作

**套利类型**：

| 套利类型 | 说明 | 风险等级 | 预期收益 |
|---------|------|---------|---------|
| 可转债套利 | 可转债与正股套利 | ⭐⭐⭐⭐⭐ | 8%-15% |
| ETF套利 | ETF与成分股套利 | ⭐⭐⭐⭐⭐ | 5%-10% |
| 期现套利 | 期货与现货套利 | ⭐⭐⭐⭐ | 10%-20% |
| 统计套利 | 统计学套利 | ⭐⭐⭐⭐ | 10%-15% |

**核心参数**：

| 参数 | 说明 | 默认值 | 调整建议 |
|-----|------|-------|---------|
| pair_threshold | 价差阈值 | 2.0 | 严格时增大 |
| entry_threshold | 入场阈值 | 1.5 | 波动大时增大 |
| exit_threshold | 出场阈值 | 0.5 | 严格出场时减小 |
| max_position | 最大仓位 | 0.2 | 大资金增加 |

**配置示例**：

```python

    'entry_threshold': 1.5,
    'exit_threshold': 0.5,
    'max_position': 0.2

```

**操作步骤**：

1. **初始化策略**
   ```python
   strategy = StatisticalArbitrageStrategy(
       pair_threshold=2.0,
       entry_threshold=1.5
   )
   ```

2. **运行回测**
   ```python
   result = engine.run_backtest(strategy)
   ```

### 6. 事件驱动策略

**适用场景**：
- 重大事件
- 政策变化
- 企业并购

**事件类型**：

| 事件类型 | 说明 | 持仓周期 | 预期收益 |

| 政策事件 | 政策发布 | 3-7 | 5%-20% |
| 并购事件 | 企业并购 | 10-20 | 15%-40% |

## 策略组合

### 组合原则

1. **风险分散**：不同风险等级的策略组合
2. **市场适应**：不同市场环境的策略组合
3. **时间互补**：不同持仓周期的策略组合

### 组合示例

```python
from core.backtest.engine import Portfolio

# 创建组合
portfolio = Portfolio(
    initial_cash=1000000,
    strategies=[
        ('trend', TrendFollowingStrategy(), 0.4),
        ('mean_rev', MeanReversionStrategy(), 0.3),
        ('momentum', MomentumStrategy(), 0.3)
    ]
)

# 运行回测
result = portfolio.run_backtest(data)
```

| 稳健型 | 30% | 30% | 20% | 20% |
| 平衡型 | 40% | 20% | 20% | 20% |

## 常见问题

### Q1: 如何选择合适的策略？

**A**：根据以下因素选择：

1. **市场环境**：牛市选趋势，震荡选均值回归
2. **风险偏好**：保守选均值回归，激进选动量
3. **资金规模**：小资金选均值回归，大资金选多因子
4. **时间精力**：时间少选趋势跟踪，时间多选多因子

### Q2: 如何优化策略参数？

**A**：使用网格搜索或遗传算法：

```python
# 网格搜索
param_grid = {
    'ma_short': [5, 10, 20],
    'ma_long': [20, 30, 50]
}
search = GridSearch(TrendFollowingStrategy, param_grid)
best_params = search.search(data)

# 遗传算法（更高效）
from core.backtest.engine import GeneticSearch
search = GeneticSearch(TrendFollowingStrategy, param_grid)
best_params = search.search(data)
```

### Q3: 如何评估策略有效性？

**A**：使用以下指标：

1. **夏普比率 > 1.0**：风险调整收益良好
2. **最大回撤 < 20%**：风险可控
3. **胜率 > 50%**：策略稳定
4. **盈亏比 > 1.5**：盈利空间充足
5. **样本外表现**：测试集表现良好

### Q4: 如何组合多个策略？

**A**：使用组合策略：

```python
portfolio = Portfolio(
    initial_cash=1000000,
    strategies=[
        ('trend', TrendFollowingStrategy(), 0.4),
        ('mean_rev', MeanReversionStrategy(), 0.3),
        ('momentum', MomentumStrategy(), 0.3)
    ]
)
result = portfolio.run_backtest(data)
```

### Q5: 如何实盘应用？

**A**：步骤：

1. **历史回测**：验证策略有效性
2. **模拟交易**：模拟实盘环境
3. **小资金试水**：小资金实盘
4. **逐步加仓**：根据表现逐步增加资金
