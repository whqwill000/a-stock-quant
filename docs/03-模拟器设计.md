## 1. 概述

### 1.1 设计目标

交易模拟器旨在模拟真实的 A 股交易环境，为策略回测和模拟交易提供逼真的交易场景。

**核心目标**：
- ✅ 准确模拟 A 股交易规则（T+1、涨跌停等）
- ✅ 提供完整的账户和订单管理
- ✅ 实现真实的撮合逻辑
- ✅ 完善的风险控制机制
- ✅ 高性能、低延迟
- ✅ 易于扩展和使用

### 1.2 模拟范围

| 模拟内容 | 说明 |
|---------|------|
| 交易规则 | T+1、涨跌停、最小交易单位 |
| 交易成本 | 印花税、佣金、过户费 |
| 订单类型 | 限价委托、市价委托 |
| 撮合逻辑 | 价格优先、时间优先 |
| 资金管理 | 可用资金、冻结资金 |
| 持仓管理 | 可用持仓、冻结持仓、持仓成本 |

### 1.3 不模拟内容

| 不模拟内容 | 说明 |
|-----------|------|
| 融资融券 | 信用交易 |
| 期权期货 | 衍生品交易 |
| 港股通 | 港股交易 |
| 新股申购 | 打新业务 |
| 分红派息 | 自动处理，不模拟流程 |

## 3. 账户管理模块

### 3.1 功能说明

账户管理模块负责管理用户的资金和持仓，是模拟器的核心组件。

### 3.2 资金管理

```python
class CapitalManager:
    def __init__(self, initial_cash: float):
        self.available_cash = initial_cash  # 可用资金
        self.frozen_cash = 0.0              # 冻结资金
        
    def freeze_cash(self, amount: float) -> bool:
        """冻结资金（下单时）"""
        if self.available_cash >= amount:
            self.available_cash -= amount
            self.frozen_cash += amount
            return True
        return False
    
    def unfreeze_cash(self, amount: float):
        """解冻资金（撤单时）"""
        self.frozen_cash -= amount
        self.available_cash += amount
        
    def deduct_cash(self, amount: float):
        """扣除资金（成交时）"""
        self.frozen_cash -= amount
        
    def add_cash(self, amount: float):
        """增加资金（卖出成交时）"""
        self.available_cash += amount
```

### 3.3 持仓管理

```python
class PositionManager:
    def __init__(self):
        self.positions: Dict[str, Position] = {}
        
    def buy(self, stock_code: str, volume: int, price: float):
        """买入建仓/加仓"""
        if stock_code not in self.positions:
            self.positions[stock_code] = Position(stock_code, 0, 0, 0, 0, 0)
        
        pos = self.positions[stock_code]
        # 计算新的持仓成本（加权平均）
        total_cost = pos.avg_cost * pos.total_volume + price * volume
        pos.total_volume += volume
        pos.avg_cost = total_cost / pos.total_volume
        # T+1: 买入的份额当日不可用
        pos.frozen_volume += volume
        
    def sell(self, stock_code: str, volume: int, price: float):
        """卖出减仓"""
        pos = self.positions[stock_code]
        pos.total_volume -= volume
        # 卖出实现盈亏
        realized_pnl = (price - pos.avg_cost) * volume
        
    def freeze(self, stock_code: str, volume: int):
        """冻结持仓（卖出委托时）"""
        pos = self.positions[stock_code]
        pos.frozen_volume += volume
        pos.available_volume -= volume
        
    def unfreeze(self, stock_code: str, volume: int):
        """解冻持仓（撤单时）"""
        pos = self.positions[stock_code]
        pos.frozen_volume -= volume
        pos.available_volume += volume
```

```python
class OrderManager:
    def __init__(self):
        self.orders: Dict[str, Order] = {}
        self.trades: List[Trade] = []
        self.order_counter = 0
        
    def create_order(self, stock_code: str, direction: str, 
                     order_type: str, price: float, volume: int) -> Order:
        """创建订单"""
        self.order_counter += 1
        order_id = f"ORD{datetime.now().strftime('%Y%m%d%H%M%S')}{self.order_counter:06d}"
        
        order = Order(
            order_id=order_id,
            stock_code=stock_code,
            direction=direction,
            order_type=order_type,
            price=price,
            volume=volume,
            filled_volume=0,
            avg_price=0.0,
            status=Order.STATUS_PENDING,
            timestamp=datetime.now()
        )
        
        self.orders[order_id] = order
        return order
        
    def submit_order(self, order_id: str) -> bool:
        """提交订单"""
        order = self.orders.get(order_id)
        if not order:
            return False
        order.status = Order.STATUS_SUBMITTED
        return True
        
    def cancel_order(self, order_id: str) -> bool:
        """撤销订单"""
        order = self.orders.get(order_id)
        if not order or order.status in [Order.STATUS_FILLED, Order.STATUS_CANCELLED]:
            return False
        order.status = Order.STATUS_CANCELLED
        return True
        
    def update_order(self, order_id: str, filled_volume: int, price: float):
        """更新订单成交信息"""
        order = self.orders[order_id]
        order.filled_volume = filled_volume
        order.avg_price = price
        
        if filled_volume >= order.volume:
            order.status = Order.STATUS_FILLED
        elif filled_volume > 0:
            order.status = Order.STATUS_PARTIAL
```

## 5. 撮合引擎模块

### 5.1 功能说明

撮合引擎是模拟器的核心，负责根据市场数据和订单队列模拟成交。

### 5.2 撮合原则

```
A 股撮合原则：
1. 价格优先：高价买入优先，低价卖出优先
2. 时间优先：同价位先申报优先
3. 市价优先：市价单优先于限价单
```

### 5.3 撮合引擎实现

```python
class MatchingEngine:
    def __init__(self):
        self.buy_orders: List[Order] = []   # 买入订单队列
        self.sell_orders: List[Order] = []  # 卖出订单队列
        
    def add_order(self, order: Order):
        """添加订单到队列"""
        if order.direction == "buy":
            self.buy_orders.append(order)
        else:
            self.sell_orders.append(order)
            
    def match(self, market_data: MarketData) -> List[Trade]:
        """执行撮合"""
        trades = []
        
        # 获取当前市场价格
        current_price = market_data.current_price
        high_price = market_data.high
        low_price = market_data.low
        
        # 处理买入订单
        for buy_order in sorted(self.buy_orders, key=lambda x: -x.price):
            if buy_order.status != Order.STATUS_SUBMITTED:
                continue
                
            # 检查是否可成交
            if self._can_match_buy(buy_order, current_price, low_price):
                trade = self._execute_buy_trade(buy_order, current_price)
                trades.append(trade)
                
        # 处理卖出订单
        for sell_order in sorted(self.sell_orders, key=lambda x: x.price):
            if sell_order.status != Order.STATUS_SUBMITTED:
                continue
                
            # 检查是否可成交
            if self._can_match_sell(sell_order, current_price, high_price):
                trade = self._execute_sell_trade(sell_order, current_price)
                trades.append(trade)
                
        return trades
        
    def _can_match_buy(self, order: Order, current: float, low: float) -> bool:
        """判断买入订单是否可成交"""
        if order.order_type == OrderType.MARKET:
            return True
        # 限价单：委托价 >= 当前最低价
        return order.price >= low
        
    def _can_match_sell(self, order: Order, current: float, high: float) -> bool:
        """判断卖出订单是否可成交"""
        if order.order_type == OrderType.MARKET:
            return True
        # 限价单：委托价 <= 当前最高价
        return order.price <= high
        
    def _execute_buy_trade(self, order: Order, price: float) -> Trade:
        """执行买入成交"""
        # 创建成交记录
        trade = Trade(
            trade_id=self._generate_trade_id(),
            order_id=order.order_id,
            stock_code=order.stock_code,
            direction="buy",
            price=price,
            volume=order.volume - order.filled_volume,
            commission=self._calculate_commission(price, order.volume),
            stamp_tax=0,  # 买入不收印花税
            transfer_fee=self._calculate_transfer_fee(order.volume),
            timestamp=datetime.now()
        )
        
        # 更新订单状态
        order.filled_volume = order.volume
        order.avg_price = price
        order.status = Order.STATUS_FILLED
        
        return trade
        
    def _execute_sell_trade(self, order: Order, price: float) -> Trade:
        """执行卖出成交"""
        volume = order.volume - order.filled_volume
        
        trade = Trade(
            trade_id=self._generate_trade_id(),
            order_id=order.order_id,
            stock_code=order.stock_code,
            direction="sell",
            price=price,
            volume=volume,
            commission=self._calculate_commission(price, volume),
            stamp_tax=self._calculate_stamp_tax(price, volume),  # 卖出收印花税
            transfer_fee=self._calculate_transfer_fee(volume),
            timestamp=datetime.now()
        )
        
        order.filled_volume = order.volume
        order.avg_price = price
        order.status = Order.STATUS_FILLED
        
        return trade
```

### 5.4 涨跌停限制

```python
class PriceLimitRule:
    # 涨跌停幅度
    LIMIT_MAIN = 0.10       # 主板 10%
    LIMIT_STAR = 0.20       # 科创板/创业板 20%
    LIMIT_ST = 0.05         # ST 股 5%
    
    def __init__(self):
        self.prev_close: Dict[str, float] = {}  # 昨日收盘价
        
    def get_limit_price(self, stock_code: str) -> Tuple[float, float]:
        """获取涨跌停价格"""
        prev_close = self.prev_close.get(stock_code, 0)
        limit_ratio = self._get_limit_ratio(stock_code)
        
        upper_limit = prev_close * (1 + limit_ratio)
        lower_limit = prev_close * (1 - limit_ratio)
        
        return round(upper_limit, 2), round(lower_limit, 2)
        
    def _get_limit_ratio(self, stock_code: str) -> float:
        """根据股票代码获取涨跌停幅度"""
        if stock_code.startswith('688') or stock_code.startswith('300'):
            return self.LIMIT_STAR
        elif stock_code.startswith('ST'):
            return self.LIMIT_ST
        else:
            return self.LIMIT_MAIN
            
    def is_limit_up(self, stock_code: str, price: float) -> bool:
        """是否涨停"""
        upper, _ = self.get_limit_price(stock_code)
        return price >= upper
        
    def is_limit_down(self, stock_code: str, price: float) -> bool:
        """是否跌停"""
        _, lower = self.get_limit_price(stock_code)
        return price <= lower
```

### 5.5 交易成本计算

```python
class CostCalculator:
    # 费率配置
    COMMISSION_RATE = 0.00025    # 佣金：万分之 2.5
    MIN_COMMISSION = 5.0         # 最低佣金：5 元
    STAMP_TAX_RATE = 0.0005      # 印花税：0.05%（卖出收取）
    TRANSFER_FEE_RATE = 0.00001  # 过户费：十万分之一
    
    def calculate_commission(self, amount: float) -> float:
        """计算佣金"""
        commission = amount * self.COMMISSION_RATE
        return max(commission, self.MIN_COMMISSION)
        
    def calculate_stamp_tax(self, amount: float) -> float:
        """计算印花税"""
        return amount * self.STAMP_TAX_RATE
        
    def calculate_transfer_fee(self, volume: int) -> float:
        """计算过户费（按股数）"""
        return volume * 0.00001  # 每股 0.00001 元
```

## 6. 风控模块

### 6.1 功能说明

风控模块负责交易风险控制，包括仓位控制、资金控制、交易限制等。

### 6.2 风控规则

```python
class RiskControl:
    def __init__(self):
        # 仓位限制
        self.max_position_ratio = 0.30      # 单只股票最大仓位 30%
        self.max_total_position = 0.95      # 最大总仓位 95%
        
        # 交易限制
        self.max_order_value = 1000000      # 单笔最大委托金额
        self.max_daily_turnover = 5000000   # 单日最大成交额
        
        # 止损限制
        self.max_loss_ratio = 0.10          # 最大亏损比例 10%
        
    def check_buy_order(self, order: Order, account: Account) -> Tuple[bool, str]:
        """检查买入订单"""
        # 1. 检查资金是否充足
        required_cash = order.price * order.volume
        if required_cash > account.available_cash:
            return False, "资金不足"
            
        # 2. 检查单笔委托金额
        if required_cash > self.max_order_value:
            return False, "超过单笔最大委托金额"
            
        # 3. 检查单只股票仓位
        current_position = account.get_position(order.stock_code)
        new_position_value = (current_position.total_volume + order.volume) * order.price
        total_asset = account.total_asset
        if new_position_value / total_asset > self.max_position_ratio:
            return False, "超过单只股票最大仓位"
            
        # 4. 检查总仓位
        new_market_value = account.market_value + required_cash
        if new_market_value / total_asset > self.max_total_position:
            return False, "超过最大总仓位"
            
        return True, "通过"
        
    def check_sell_order(self, order: Order, account: Account) -> Tuple[bool, str]:
        """检查卖出订单"""
        # 1. 检查持仓是否充足
        position = account.get_position(order.stock_code)
        if position.available_volume < order.volume:
            return False, "可用持仓不足"
            
        return True, "通过"
        
    def check_stop_loss(self, stock_code: str, current_price: float, 
                        position: Position) -> Tuple[bool, float]:
        """检查止损"""
        if position.avg_cost == 0:
            return False, 0
            
        loss_ratio = (current_price - position.avg_cost) / position.avg_cost
        if loss_ratio < -self.max_loss_ratio:
            return True, loss_ratio
            
        return False, loss_ratio
```

```python
        """日终处理"""
        # 1. 更新持仓市值
        self._update_market_value()
        
        # 2. 处理 T+1 到期
        self._process_t_plus1(trade_date)
        
        # 3. 计算当日盈亏
        self._calculate_daily_pnl()
        
        # 4. 清理过期数据
        self._cleanup_old_data()
        
    def _process_t_plus1(self, trade_date: date):
        """处理 T+1 到期：昨日买入的今日可用"""
```

## 9. 使用示例

### 9.1 基本使用

```python
from simulator import TradingSimulator

# 1. 创建模拟器（初始资金 100 万）
simulator = TradingSimulator(initial_cash=1000000)

# 2. 买入股票
order_id = simulator.buy(
    stock_code="000001.SZ",
    price=10.50,
    volume=1000,  # 10 手
    order_type="limit"
)

# 3. 查询订单
order = simulator.get_order(order_id)
print(f"订单状态：{order.status}")

# 4. 喂入市场数据（触发撮合）
market_data = MarketData(
    stock_code="000001.SZ",
    current_price=10.50,
    high=10.60,
    low=10.40
)
simulator.feed_market_data(market_data)

# 5. 查询持仓
position = simulator.get_position("000001.SZ")
print(f"持仓数量：{position.total_volume}")
print(f"持仓成本：{position.avg_cost}")

# 6. 卖出股票
sell_order_id = simulator.sell(
    stock_code="000001.SZ",
    price=11.00,
    volume=500
)

# 7. 查询账户
account = simulator.get_account()
print(f"总资产：{account.total_asset}")
print(f"可用资金：{account.available_cash}")
```
