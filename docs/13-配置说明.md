## 全局配置

配置文件：`config/default.yaml`

### 应用配置

```yaml
# 应用配置
app:
  name: "A 股量化金融平台"      # 应用名称
  version: "0.1.0"             # 版本号
  debug: false                 # 调试模式（true/false）
  log_level: "INFO"            # 日志级别（DEBUG/INFO/WARNING/ERROR）
```

**配置说明**：

| 参数 | 说明 | 可选值 | 默认值 |
|-----|------|-------|-------|
| name | 应用名称 | 字符串 | "A 股量化金融平台" |
| version | 版本号 | 字符串 | "0.1.0" |
| debug | 调试模式 | true/false | false |
| log_level | 日志级别 | DEBUG/INFO/WARNING/ERROR | INFO |

### 数据配置

```yaml
# 数据配置
data:
  # 默认数据源
  default_source: "akshare"     # 数据源选择：akshare/tushare

  # 数据缓存
  cache:
    enabled: true               # 是否启用缓存
    directory: "data/cache"     # 缓存目录
    expire_hours: 24            # 缓存过期时间（小时）

  # 股票池配置
  stock_pool:
    exclude_st: true            # 排除 ST 股票
    exclude_new: true           # 排除新股（上市<60日）
    min_market_cap: 1000000000  # 最小市值（元）
    min_liquidity: 10000000     # 最小日成交额（元）
```

**配置说明**：

| 参数 | 说明 | 可选值 | 默认值 |
|-----|------|-------|-------|
| default_source | 默认数据源 | akshare/tushare | akshare |
| cache.enabled | 启用缓存 | true/false | true |
| cache.directory | 缓存目录 | 路径 | data/cache |
| cache.expire_hours | 缓存过期时间 | 整数 | 24 |
| stock_pool.exclude_st | 排除ST股 | true/false | true |
| stock_pool.exclude_new | 排除新股 | true/false | true |
| stock_pool.min_market_cap | 最小市值 | 整数 | 1000000000 |
| stock_pool.min_liquidity | 最小成交额 | 整数 | 10000000 |

### 交易配置

```yaml
# 交易配置
trading:
  # A 股交易规则
  t_plus_1: true                # T+1 交易（true/false）
  price_limit: true             # 涨跌停限制（true/false）

  # 交易成本
  cost:
    commission: 0.00025         # 佣金费率（万2.5）
    min_commission: 5.0         # 最低佣金（元）
    stamp_tax: 0.0005           # 印花税率（0.05%）
    transfer_fee: 0.00001       # 过户费率（十万分之一）
    slippage: 0.001             # 滑点（0.1%）

  # 风控配置
  risk_control:
    max_position_ratio: 0.30    # 单只股票最大仓位（30%）
    max_total_position: 0.95    # 最大总仓位（95%）
    max_loss_ratio: 0.10        # 最大亏损比例（10%）
    stop_loss_threshold: 0.05   # 止损阈值（5%）
    take_profit_threshold: 0.15 # 止盈阈值（15%）
```

**配置说明**：

| 参数 | 说明 | 可选值 | 默认值 |
|-----|------|-------|-------|
| t_plus_1 | T+1 交易 | true/false | true |
| price_limit | 涨跌停限制 | true/false | true |
| cost.commission | 佣金费率 | 浮点数 | 0.00025 |
| cost.min_commission | 最低佣金 | 浮点数 | 5.0 |
| cost.stamp_tax | 印花税率 | 浮点数 | 0.0005 |
| cost.transfer_fee | 过户费率 | 浮点数 | 0.00001 |
| cost.slippage | 滑点 | 浮点数 | 0.001 |
| risk_control.max_position_ratio | 单股最大仓位 | 0.0-1.0 | 0.30 |
| risk_control.max_total_position | 最大总仓位 | 0.0-1.0 | 0.95 |
| risk_control.max_loss_ratio | 最大亏损比例 | 0.0-1.0 | 0.10 |
| risk_control.stop_loss_threshold | 止损阈值 | 0.0-1.0 | 0.05 |
| risk_control.take_profit_threshold | 止盈阈值 | 0.0-1.0 | 0.15 |

# 趋势跟踪策略配置
strategy:
  name: "TrendFollowing"        # 策略名称
  description: "趋势跟踪策略"    # 策略描述

  # 参数配置
  params:
    ma_short: 5                 # 短期均线周期
    ma_long: 20                 # 长期均线周期
    entry_threshold: 0.02       # 入场阈值（2%）
    exit_threshold: 0.05        # 出场阈值（5%）
    stop_loss: 0.08             # 止损阈值（8%）
    take_profit: 0.15           # 止盈阈值（15%）

  # 交易配置
  trading:
    position_ratio: 0.3         # 仓位比例（30%）
    holding_period: 20          # 持仓周期（交易日）
    min_volume: 1000000         # 最小成交量

  # 市场适应配置
  market_adaptation:
    bull:
      position_ratio: 0.7       # 牛市仓位
      ma_short: 5
      ma_long: 20
    bear:
      position_ratio: 0.3       # 熊市仓位
      ma_short: 5
      ma_long: 20
    ranging:
      position_ratio: 0.1       # 震荡市仓位
      ma_short: 10
      ma_long: 30
```

    directory: "data/cache"
    expire_hours: 24
```

logging:
  level: "DEBUG"  # 调试模式
