## 1. 趋势跟踪策略

### 1.1 策略原理

**核心思想**：顺势而为，价格沿趋势方向运动

```
基本假设：
1. 市场存在趋势（非完全随机游走）
2. 趋势一旦形成，倾向于延续
3. 在趋势结束前入场，在趋势反转前离场
```

### 1.2 理论基础

**道氏理论**：
- 主要趋势（长期）：持续数月至数年
- 次要趋势（中期）：持续数周至数月
- 短暂趋势（短期）：持续数小时至数周

**趋势类型**：
```
上升趋势：高点不断抬高，低点也不断抬高
    ╱╲    ╱╲
   ╱  ╲  ╱  ╲
  ╱    ╲╱    ╲

下降趋势：低点不断降低，高点也不断降低
  ╲    ╱╲    ╱
   ╲  ╱  ╲  ╱
    ╲╱    ╲╱

横盘震荡：价格在区间内波动
  ───────────
   ╲╱╲╱╲╱╲╱
  ───────────
```

### 1.3 常用指标

#### 1.3.1 移动平均线 (MA)

```python
# 计算公式
MA(N) = (P1 + P2 + ... + PN) / N

# 常见周期
MA5:  5 日均线（周线）
MA10: 10 日均线（半月线）
MA20: 20 日均线（月线）
MA60: 60 日均线（季线）
MA120: 120 日均线（半年线）
MA250: 250 日均线（年线）
```

**交易信号**：
| 信号类型 | 条件 | 操作 |
|---------|------|------|
| 金叉 | 短期 MA 上穿长期 MA | 买入 |
| 死叉 | 短期 MA 下穿长期 MA | 卖出 |
| 多头排列 | 短期 MA > 中期 MA > 长期 MA | 持有 |
| 空头排列 | 短期 MA < 中期 MA < 长期 MA | 空仓 |

#### 1.3.2 MACD (Moving Average Convergence Divergence)

```python
# 计算步骤
1. 计算 12 日 EMA（快线）
2. 计算 26 日 EMA（慢线）
3. DIF = EMA12 - EMA26
4. DEA = DIF 的 9 日 EMA（信号线）
5. MACD 柱 = (DIF - DEA) × 2

# 交易信号
- DIF 上穿 DEA：金叉，买入
- DIF 下穿 DEA：死叉，卖出
- MACD 柱由负转正：买入
- MACD 柱由正转负：卖出
```

#### 1.3.3 布林带 (Bollinger Bands)

```python
# 计算
中轨 = MA20
上轨 = 中轨 + 2 × Std(20)
下轨 = 中轨 - 2 × Std(20)

# 趋势判断
- 价格突破上轨：强势上涨
- 价格跌破下轨：弱势下跌
- 带宽收窄：即将变盘
- 带宽扩张：趋势加强
```

```
┌─────────────────────────────────────────────────────────────┐
│                    趋势跟踪策略流程                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 数据准备                                                │
│     └─→ 获取历史行情数据（日 K 线）                          │
│                                                             │
│  2. 指标计算                                                │
│     └─→ 计算 MA/MACD/布林带等指标                           │
│                                                             │
│  3. 信号生成                                                │
│     └─→ 根据指标交叉/突破生成买卖信号                        │
│                                                             │
│  4. 信号过滤                                                │
│     └─→ 成交量确认、趋势强度过滤                            │
│                                                             │
│  5. 仓位管理                                                │
│     └─→ 根据信号强度确定仓位比例                            │

```

### 1.5 A 股适配要点

| 要点 | 说明 |
|------|------|
| **T+1 限制** | 当日买入无法当日卖出，需隔夜持仓 |
| **涨跌停** | 涨停买不进，跌停卖不出，需特殊处理 |
| **交易成本** | 考虑印花税、佣金、滑点 |
| **停牌处理** | 停牌期间无法交易，需跳过 |
| **除权除息** | 使用前复权数据，避免价格跳空 |

### 1.6 优缺点分析

| 优点 | 缺点 |
|------|------|
| 逻辑简单，易于理解 | 震荡市反复止损 |
| 趋势行情收益丰厚 | 信号滞后 |
| 可交易多个标的 | 需要耐心等待趋势 |
| 不需要预测市场 | 胜率通常低于 50% |

### 1.7 适用场景

- ✅ 牛市/熊市明显阶段
- ✅ 强势板块/龙头股
- ❌ 震荡市、无明显方向

## 2. 均值回归策略

### 2.1 策略原理

**核心思想**：价格围绕价值波动，偏离均值后会回归

```
基本假设：
1. 价格存在"合理"区间
2. 过度偏离会被纠正
3. 涨多了会跌，跌多了会涨
```

### 2.2 理论基础

**统计学原理**：
- 大数定律：长期来看，价格趋向于均值
- 正态分布：大部分数据集中在均值附近

**行为金融学解释**：
- 过度反应：投资者对消息反应过度，导致价格超调
- 获利回吐：大幅上涨后，获利盘卖出导致回调

### 2.3 常用指标

#### 2.3.1 布林带回归

```python
# 计算
中轨 = MA20
上轨 = 中轨 + 2 × Std(20)
下轨 = 中轨 - 2 × Std(20)

# 交易信号
- 价格触及下轨 → 买入（超卖）
- 价格触及上轨 → 卖出（超买）
- 价格回归中轨 → 平仓
```

#### 2.3.2 RSI (Relative Strength Index)

```python
# 计算
RS = N 日内平均涨幅 / N 日内平均跌幅
RSI = 100 - 100 / (1 + RS)

# 常见周期：N=6, 12, 24

# 交易信号
- RSI < 30: 超卖，买入
- RSI > 70: 超买，卖出
- RSI 从超卖区回升：买入
- RSI 从超买区回落：卖出
```

#### 2.3.3 KDJ 指标

```python
# 计算
RSV = (收盘价 - N 日最低价) / (N 日最高价 - N 日最低价) × 100
K = RSV 的 M 日 SMA
D = K 的 M 日 SMA
J = 3 × K - 2 × D

# 交易信号
- K/D < 20: 超卖，买入
- K/D > 80: 超买，卖出
- J > 100: 严重超买
- J < 0: 严重超卖
```

#### 2.3.4 价格偏离度

```python
# 计算价格与均线的偏离程度
偏离度 = (当前价格 - N 日均线) / N 日均线

# 交易信号
- 偏离度 < -阈值：买入（价格过低）
- 偏离度 > 阈值：卖出（价格过高）
```

### 2.4 策略实现流程

```
┌─────────────────────────────────────────────────────────────┐
│                  均值回归策略流程                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 选择标的                                                │
│     └─→ 选择波动性适中、流动性好的股票                      │
│                                                             │
│  2. 计算指标                                                │
│     └─→ 计算布林带/RSI/KDJ 等指标                           │
│                                                             │
│  3. 确定阈值                                                │
│     └─→ 根据历史数据确定超买超卖阈值                        │
│                                                             │
│  4. 生成信号                                                │
│     └─→ 触及阈值生成反向交易信号                            │
│                                                             │
│  5. 设置目标                                                │
│     └─→ 以均值作为目标价位                                  │
│                                                             │
│  6. 止损设置                                                │
│     └─→ 防止价格继续偏离，设置止损                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.5 A 股适配要点

| 要点 | 说明 |
|------|------|
| **波动性选择** | 选择波动适中的股票，避免涨停/跌停无法交易 |
| **时间周期** | 日内回归较快，隔夜回归较慢 |
| **止损设置** | 防止单边行情，必须设置止损 |
| **仓位控制** | 分批建仓，避免一次性重仓 |

### 2.6 优缺点分析

| 优点 | 缺点 |
|------|------|
| 胜率较高 | 单次收益有限 |
| 适合震荡市 | 趋势市容易亏损 |
| 交易频率适中 | 需要设置严格止损 |
| 逻辑清晰 | 可能错过趋势行情 |

### 2.7 适用场景

- ✅ 震荡市、盘整期
- ✅ 波动性稳定的股票
- ❌ 单边趋势行情
- ❌ 重大消息发布前后

## 3. 多因子选股策略

### 3.1 策略原理

**核心思想**：从多个维度评估股票，综合评分选股

```
基本框架：
1. 选取多个有效因子
2. 对每个因子打分
3. 加权得到综合评分
4. 买入评分高的股票
```

### 3.2 因子类型

#### 3.2.1 价值因子

| 因子 | 计算公式 | 说明 |
|------|---------|------|
| **PE (市盈率)** | 股价 / 每股收益 | 越低越便宜 |
| **PB (市净率)** | 股价 / 每股净资产 | 越低越便宜 |
| **PS (市销率)** | 市值 / 营业收入 | 越低越便宜 |
| **股息率** | 每股股息 / 股价 | 越高越好 |

#### 3.2.2 成长因子

| 因子 | 计算公式 | 说明 |
|------|---------|------|
| **营收增速** | (本期营收 - 上期营收) / 上期营收 | 越高越好 |
| **利润增速** | (本期利润 - 上期利润) / 上期利润 | 越高越好 |
| **ROE 变化** | 本期 ROE - 上期 ROE | 提升越好 |

#### 3.2.3 动量因子

| 因子 | 计算公式 | 说明 |
|------|---------|------|
| **N 日收益率** | (今日收盘价 - N 日前收盘价) / N 日前收盘价 | 越高越强 |
| **相对强度** | 个股涨幅 / 指数涨幅 | 大于 1 为强 |
| **新高比例** | (收盘价 - N 日最低价) / (N 日最高价 - N 日最低价) | 越高越强 |

#### 3.2.4 质量因子

| 因子 | 计算公式 | 说明 |
|------|---------|------|
| **ROE** | 净利润 / 净资产 | 越高越好 |
| **毛利率** | 毛利润 / 营业收入 | 越高越好 |
| **资产负债率** | 负债 / 资产 | 适中为好 |
| **经营现金流** | 经营活动现金流 / 净利润 | 越高越好 |

#### 3.2.5 规模因子

| 因子 | 说明 |
|------|------|
| **总市值** | 总市值越小，弹性越大（小市值效应） |
| **流通市值** | 流通盘越小，越容易被炒作 |

#### 3.2.6 情绪因子

| 因子 | 说明 |
|------|------|
| **换手率** | 成交活跃度 |
| **量比** | 当前成交量 / 过去平均成交量 |
| **资金流向** | 主力净流入/流出 |

### 3.3 因子处理流程

```
┌─────────────────────────────────────────────────────────────┐
│                    因子处理流程                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 数据预处理                                              │
│     ├─→ 缺失值处理（填充/删除）                             │
│     └─→ 异常值处理（Winsorization）                         │
│                                                             │
│  2. 标准化                                                  │
│     └─→ Z-Score: (x - mean) / std                          │
│                                                             │
│  3. 中性化                                                  │
│     └─→ 去除行业、市值影响                                  │
│                                                             │
│  4. 因子加权                                                │
│     ├─→ 等权加权                                            │
│     ├─→ IC 加权（根据因子预测能力）                          │
│     └─→ 机器学习加权                                        │
│                                                             │
│  5. 综合评分                                                │
│     └─→ 计算每只股票的综合得分                              │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.4 策略实现流程

```python
# 伪代码示例
def multi_factor_strategy():
    # 1. 股票池筛选
    stock_pool = filter_stocks(
        exclude_st=True,      # 排除 ST
        exclude_new=True,     # 排除新股
        min_liquidity=1000    # 最小流动性
    )
    
    # 2. 因子计算
    factors = {
        'value': calculate_value_factor(stock_pool),
        'growth': calculate_growth_factor(stock_pool),
        'momentum': calculate_momentum_factor(stock_pool),
        'quality': calculate_quality_factor(stock_pool)
    }
    
    # 3. 因子加权
    weights = {'value': 0.3, 'growth': 0.3, 'momentum': 0.2, 'quality': 0.2}
    scores = weighted_sum(factors, weights)
    
    # 4. 选股
    selected = select_top_n(scores, n=10)
    
    # 5. 调仓
    rebalance(selected)
```

### 3.5 A 股适配要点

| 要点 | 说明 |
|------|------|
| **股票池** | 剔除 ST、*ST、停牌、新股 |
| **调仓频率** | 周度/月度，避免过于频繁 |
| **行业中性** | 控制行业暴露，避免行业集中 |
| **市值中性** | 控制市值暴露 |
| **交易成本** | 考虑调仓成本 |

### 3.6 优缺点分析

| 优点 | 缺点 |
|------|------|
| 逻辑全面，多维度选股 | 实现复杂度高 |
| 分散风险，不依赖单一因子 | 需要大量数据 |
| 资金容量大 | 需要定期维护因子 |
| 可解释性强 | 因子可能失效 |

### 3.7 适用场景

- ✅ 中长期投资
- ✅ 大资金运作
- ✅ 机构化投资
- ❌ 短线炒作

## 4. 动量策略

### 4.1 策略原理

**核心思想**：强者恒强，过去表现好的股票未来继续表现好

```
动量效应：
- 中期动量（3-12 个月）：效应最明显
- 短期动量（1-4 周）：可能存在反转
- 长期动量（1-3 年）：可能存在反转
```

### 4.2 理论基础

**行为金融学解释**：
- 反应不足：投资者对新信息反应缓慢
- 羊群效应：投资者跟随他人行为
- 处置效应：投资者过早卖出盈利股票，过晚卖出亏损股票

### 4.3 常用策略

#### 4.3.1 相对强度动量

```python
# 计算相对强度
RS = 个股 N 日收益率 / 指数 N 日收益率

# 选股
- 选择 RS 最高的前 N 只股票
- 定期（如每周）调仓
```

#### 4.3.2 价格突破动量

```python
# 突破 N 日新高
if 收盘价 > Max(过去 N 日最高价):
    买入信号

# 跌破 N 日新低
if 收盘价 < Min(过去 N 日最低价):
    卖出信号
```

#### 4.3.3 行业轮动动量

```python
# 选择强势行业
行业强度 = 行业指数 N 日收益率

# 在强势行业中选择强势个股
个股强度 = 个股 N 日收益率

# 双重动量筛选
selected = 强势行业中的强势个股
```

### 4.4 策略实现流程

```
┌─────────────────────────────────────────────────────────────┐
│                    动量策略流程                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 确定动量周期                                            │
│     └─→ 常见：20 日、60 日、120 日                           │
│                                                             │
│  2. 计算动量指标                                            │
│     └─→ 收益率、相对强度、突破信号                          │
│                                                             │
│  3. 排序选股                                                │
│     └─→ 选择动量最强的前 N 只股票                           │
│                                                             │
│  4. 设置止损                                                │
│     └─→ 动量反转时及时止损                                  │
│                                                             │
│  5. 定期调仓                                                │
│     └─→ 周度/月度调仓，保持动量暴露                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 4.5 A 股适配要点

| 要点 | 说明 |
|------|------|
| **T+1 限制** | 当日买入次日才能卖出，需注意 timing |
| **涨跌停** | 涨停股可能买不进，需备选方案 |
| **止损** | 动量反转快，需严格止损 |
| **热点轮动** | A 股热点轮动快，需及时跟踪 |

### 4.6 优缺点分析

| 优点 | 缺点 |
|------|------|
| 牛市收益丰厚 | 震荡市反复止损 |
| 逻辑简单 | 需要快速反应 |
| 可结合热点 | 可能追高 |
| 趋势延续时收益高 | 反转时亏损大 |

### 4.7 适用场景

- ✅ 牛市、强势股
- ✅ 热点题材
- ❌ 震荡市
- ❌ 弱势股

## 5. 套利策略

### 5.1 策略原理

**核心思想**：利用价格差异，获取无风险/低风险收益

```
套利条件：
1. 存在价格差异
2. 差异可以锁定
3. 交易成本低于差异
4. 可以快速执行
```

### 5.2 套利类型

#### 5.2.1 期现套利

```python
# 原理：期货与现货价格存在差异
当 期货价格 > 现货价格 + 持有成本:
    做空期货，做多现货（正向套利）

当 期货价格 < 现货价格 - 持有成本:
    做多期货，做空现货（反向套利）

# A 股限制：需要股指期货权限，门槛较高
```

#### 5.2.2 ETF 套利

```python
# 原理：ETF 价格与净值存在差异
当 ETF 价格 > IOPV（净值）:
    申购 ETF 份额 → 赎回一篮子股票 → 卖出股票

当 ETF 价格 < IOPV（净值）:
    买入一篮子股票 → 申购 ETF 份额 → 卖出 ETF

# A 股限制：需要快速通道，适合机构
```

#### 5.2.3 可转债套利

```python
# 原理：可转债与正股存在转换关系
转股价值 = 正股价格 × 转股比例

当 转债价格 < 转股价值:
    买入转债 → 转股 → 卖出正股

# A 股适用：门槛低，个人可参与
```

#### 5.2.4 并购套利

```python
# 原理：并购公告后，目标公司股价通常低于收购价
当 并购公告发布:
    买入目标公司股票
    等待并购完成获利

# 风险：并购可能失败
```

### 5.3 策略实现流程

```
┌─────────────────────────────────────────────────────────────┐
│                    套利策略流程                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 监控价差                                                │
│     └─→ 实时监控相关资产的价格差异                          │
│                                                             │
│  2. 计算理论价差                                            │
│     └─→ 考虑持有成本、交易成本等                            │
│                                                             │
│  3. 判断套利机会                                            │
│     └─→ 实际价差 > 理论价差 + 安全边际                      │
│                                                             │
│  4. 执行套利                                                │
│     └─→ 同时执行两边交易，锁定利润                          │
│                                                             │
│  5. 平仓了结                                                │
│     └─→ 价差回归后平仓                                      │

```

### 5.4 A 股适配要点

- ✅ 有专业知识和工具

## 6. 事件驱动策略

### 6.1 策略原理

**核心思想**：利用特定事件带来的价格波动获利

```
事件类型：
1. 公司事件（财报、分红、重组等）
2. 市场事件（指数调整、政策变化等）
3. 宏观事件（经济数据、货币政策等）
```

### 6.2 事件类型

#### 6.2.1 财报事件

```python
# 财报超预期
当 实际 EPS > 预期 EPS:
    买入信号（短期上涨概率大）

# 财报低于预期
当 实际 EPS < 预期 EPS:
    卖出信号（短期下跌概率大）

# 关键指标
- 营收增速
- 利润增速
- 毛利率变化
- 现金流状况
```

#### 6.2.2 分红除权事件

```python
# 抢权行情（分红公告后到除权前）
当 分红预案公告:
    买入信号（预期抢权行情）

# 填权行情（除权后）
当 除权完成:
    观察是否填权（价格上涨回补缺口）
```

#### 6.2.3 重组并购事件

```python
# 重组公告
当 重组公告发布:
    分析重组质量
    优质重组 → 买入

# 并购公告
当 并购公告发布:
    买入目标公司（通常有溢价）
```

#### 6.2.4 股权激励事件

```python
# 股权激励计划
当 股权激励公告:
    行权价通常低于当前股价
    买入信号（管理层有信心）
```

#### 6.2.5 指数调仓事件

```python
# 指数成分股调整
当 指数公司宣布调整:
    调入股票 → 买入（指数基金被动买入）
    调出股票 → 卖出（指数基金被动卖出）
```

### 6.3 策略实现流程

```
┌─────────────────────────────────────────────────────────────┐
│                  事件驱动策略流程                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 事件监控                                                │
│     └─→ 监控公告、新闻、数据发布                            │
│                                                             │
│  2. 事件识别                                                │
│     └─→ 识别事件类型和影响                                  │
│                                                             │
│  3. 影响评估                                                │
│     └─→ 评估事件对股价的影响方向和幅度                      │
│                                                             │
│  4. 交易决策                                                │
│     └─→ 根据评估结果制定交易计划                            │
│                                                             │
│  5. 执行交易                                                │
│     └─→ 快速执行，抢占先机                                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 6.4 A 股适配要点

| 要点 | 说明 |
|------|------|
| **信息获取** | 关注巨潮资讯、上交所、深交所公告 |
| **反应速度** | 重大事件需快速反应 |
| **政策解读** | A 股政策敏感，需理解政策意图 |
| **题材炒作** | A 股喜欢炒作题材，可顺势而为 |

### 6.5 优缺点分析

| 优点 | 缺点 |
|------|------|
| 收益潜力大 | 需要快速反应 |
| 与市场相关性低 | 信息获取成本高 |
| 可结合基本面 | 事件可能不及预期 |
| 机会多样 | 需要专业知识 |

### 6.6 适用场景

- ✅ 财报季
- ✅ 政策密集期
- ✅ 并购活跃期
- ❌ 信息不对称时

## 7. 资金流策略

### 7.1 策略原理

**核心思想**：跟随主力资金，资金流向决定价格走向

```
基本假设：
1. 资金是推动价格上涨的直接动力
2. 主力资金（机构、外资）更聪明
3. 资金流向可以提前预示价格变化
```

### 7.2 资金类型

#### 7.2.1 北向资金

```python
# 定义：通过沪/深股通投资 A 股的外资
特点：
- 被称为"聪明钱"
- 偏好价值投资
- 对 market 影响大

# 跟踪方式
- 每日北向资金净流入/流出
- 北向资金持仓变化
- 北向资金重仓股
```

#### 7.2.2 龙虎榜资金

```python
# 定义：交易所公布的异常交易数据
上榜条件：
- 日涨跌幅偏离值±7%
- 日换手率>20%
- 日价格振幅±15%

# 分析维度
- 买入/卖出金额
- 营业部席位
- 机构专用席位
```

#### 7.2.3 大宗交易资金

```python
# 定义：大额场外交易
特点：
- 价格通常有折价
- 买方多为机构
- 可能预示后续走势
```

#### 7.2.4 主力资金流

```python
# 定义：大单资金流向
计算方法：
- 大单：成交额>100 万或成交量>1000 手
- 主力净流入 = 大单买入 - 大单卖出

# 指标
- 主力净流入额
- 主力净流入比例
- 连续净流入天数
```

### 7.3 常用指标

| 指标 | 计算公式 | 说明 |
|------|---------|------|
| **资金净流入** | 买入金额 - 卖出金额 | 正值表示流入 |
| **资金流比例** | 净流入 / 成交额 | 标准化指标 |
| **量比** | 当前成交量 / 过去 5 日平均成交量 | 放量/缩量 |
| **换手率** | 成交量 / 流通股本 | 交易活跃度 |
| **资金集中度** | 大单成交占比 | 主力参与度 |

### 7.4 策略实现流程

```
┌─────────────────────────────────────────────────────────────┐
│                  资金流策略流程                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 资金流监控                                              │
│     └─→ 监控北向资金、龙虎榜、主力流入等                    │
│                                                             │
│  2. 资金流分析                                              │
│     └─→ 分析资金流向、持续性、强度                          │
│                                                             │
│  3. 选股筛选                                                │
│     └─→ 选择资金持续流入的股票                              │
│                                                             │
│  4. 确认信号                                                │
│     └─→ 结合价格、成交量确认                                │
│                                                             │
│  5. 跟随交易                                                │
│     └─→ 跟随主力资金方向交易                                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 7.5 A 股适配要点

| 要点 | 说明 |
|------|------|
| **北向资金** | 关注外资动向，特别是重仓股 |
| **龙虎榜** | 关注机构席位、知名游资 |
| **主力流入** | 结合价格位置判断 |
| **假流入** | 警惕主力对倒出货 |

### 7.6 优缺点分析

| 优点 | 缺点 |
|------|------|
| 跟随聪明钱 | 数据可能滞后 |
| 逻辑直观 | 可能接盘 |
| 可结合其他策略 | 需要辨别真假 |
| 适合短线 | 不适合长线 |

### 7.7 适用场景

- ✅ 短线交易
- ✅ 题材炒作
- ✅ 跟随外资
- ❌ 数据不准确时

## 8. 策略对比与选择

### 8.1 策略对比表

| 策略 | 难度 | 资金需求 | 预期年化 | 最大回撤 | 适合市场 | 持仓周期 |
|------|------|---------|---------|---------|---------|---------|
| 趋势跟踪 | ⭐⭐ | 低 | 20%-50% | 20%-30% | 趋势市 | 周 - 月 |
| 均值回归 | ⭐⭐ | 低 | 15%-40% | 15%-25% | 震荡市 | 日 - 周 |
| 多因子选股 | ⭐⭐⭐⭐ | 中 | 15%-30% | 10%-20% | 所有 | 月 - 季 |
| 动量策略 | ⭐⭐ | 低 | 20%-50% | 20%-35% | 牛市 | 周 - 月 |
| 套利策略 | ⭐⭐⭐⭐⭐ | 高 | 8%-20% | 5%-15% | 所有 | 日 - 周 |
| 事件驱动 | ⭐⭐⭐ | 中 | 15%-40% | 15%-30% | 所有 | 日 - 月 |
| 资金流策略 | ⭐⭐ | 低 | 20%-50% | 20%-35% | 所有 | 日 - 周 |

### 8.2 策略选择建议

#### 按资金规模选择

| 资金规模 | 推荐策略 |
|---------|---------|
| < 10 万 | 趋势跟踪、均值回归、动量策略 |
| 10-100 万 | 多因子选股、事件驱动、资金流策略 |
| 100-1000 万 | 多因子选股、市场中性策略 |
| > 1000 万 | 复合策略、套利策略 |

#### 按交易风格选择

| 交易风格 | 推荐策略 |
|---------|---------|
| 短线交易 | 均值回归、资金流策略 |
| 中线交易 | 趋势跟踪、动量策略 |
| 长线投资 | 多因子选股 |
| 低风险偏好 | 套利策略、多因子选股 |
| 高风险偏好 | 趋势跟踪、动量策略 |

#### 按市场环境选择

| 市场环境 | 推荐策略 |
|---------|---------|
| 牛市 | 趋势跟踪、动量策略 |
| 熊市 | 均值回归、套利策略 |
| 震荡市 | 均值回归、事件驱动 |
| 不确定 | 多因子选股（分散风险） |

### 8.3 策略组合建议

```
推荐组合：

1. 核心 - 卫星策略
   核心（60%）：多因子选股（稳定收益）
   卫星（40%）：趋势跟踪 + 动量策略（增强收益）

2. 多策略组合
   趋势跟踪（30%）+ 均值回归（30%）+ 多因子（40%）
   不同策略相关性低，分散风险

3. 市场中性策略
   多头（多因子选股）+ 空头（股指期货对冲）
   获取纯 Alpha 收益
```
