## 快速开始

```bash
# 1. 克隆项目
cd /home/whqwill/code
git clone https://github.com/your-username/a-stock-quant.git
cd a-stock-quant

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行示例
python examples/simple_backtest.py
```

## 系统要求

### 最低配置
- 操作系统：Linux (Ubuntu 20.04+) / macOS / Windows 10+
- Python：3.10+
- 内存：8GB
- 存储：20GB 可用空间
- 网络：稳定网络连接

### 推荐配置
- 操作系统：Linux (Ubuntu 22.04+) / macOS 12+
- Python：3.10-3.12
- 内存：16GB+
- 存储：50GB+ SSD
- 网络：稳定网络连接

### GPU 配置（可选）
- GPU：NVIDIA RTX 3060+ (6GB+ 显存)
- CUDA：11.8 或 12.8
- 驱动：525+

## 部署步骤

### 步骤 1：环境准备

#### 检查 Python 版本

```bash
python --version
# 应显示：Python 3.10+ 或 Python 3.12+

# 如果需要安装 Python 3.12
# Ubuntu
sudo apt update
sudo apt install python3.12 python3.12-venv python3.12-dev

# macOS
brew install python@3.12
```

#### 创建虚拟环境（推荐）

```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Linux/Mac
source venv/bin/activate

# Windows
venv\Scripts\activate

# 验证虚拟环境
which python
# 应显示：/path/to/a-stock-quant/venv/bin/python
```

### 步骤 2：克隆项目

```bash
# 进入工作目录
cd /home/whqwill/code

# 克隆项目
git clone https://github.com/your-username/a-stock-quant.git
cd a-stock-quant

# 验证项目结构
ls -la
```

### 步骤 3：安装依赖

#### 方式一：使用 pip

```bash
# 安装基础依赖
pip install -r requirements.txt

# 或安装所有依赖（包括可选依赖）
pip install -r requirements.txt --extra-index-url https://download.pytorch.org/whl/cu128
```

#### 方式二：使用 conda

```bash
# 创建 conda 环境
conda create -n a-stock-quant python=3.12

# 激活环境
conda activate a-stock-quant

# 安装依赖
pip install -r requirements.txt
```

#### 方式三：分步安装

```bash
# 1. 安装数据处理
pip install pandas numpy polars

# 2. 安装数据获取
pip install akshare tushare

# 3. 安装机器学习（可选）
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu128
pip install lightgbm xgboost catboost

# 4. 安装可视化
pip install matplotlib plotly seaborn

# 5. 安装其他工具
pip install pyyaml python-dotenv tqdm joblib
```

### 步骤 4：配置环境

#### 创建配置目录

```bash
# 创建数据目录
mkdir -p data/raw
mkdir -p data/processed
mkdir -p data/cache

# 创建日志目录
mkdir -p logs
```

#### 配置数据源（可选）

```bash
# 编辑配置文件
vim config/default.yaml

# 或使用默认配置
# 默认配置已包含常用设置
```

### 步骤 5：验证安装

#### 运行环境检查脚本

```bash
# 运行环境检查
python scripts/check_env.py
```

#### 运行测试

```bash
# 运行所有测试
pytest tests/ -v

# 或运行特定测试
pytest tests/test_utils.py -v
pytest tests/test_simulator.py -v
pytest tests/test_backtest.py -v
```

#### 运行示例

```bash
# 运行回测示例
python examples/simple_backtest.py
```

## 配置说明

### 全局配置

配置文件位置：`config/default.yaml`

```yaml
# 全局配置文件

# 应用配置
app:
  name: "A 股量化金融平台"
  version: "0.1.0"
  debug: false

# 数据配置
data:
  default_source: "akshare"
  cache:
    enabled: true
    directory: "data/cache"
    expire_hours: 24

# 交易配置
trading:
  t_plus_1: true
  price_limit: true
  cost:
    commission: 0.00025
    min_commission: 5.0
    stamp_tax: 0.0005
    transfer_fee: 0.00001
    slippage: 0.001
  risk_control:
    max_position_ratio: 0.30
    max_total_position: 0.95
    max_loss_ratio: 0.10

# 回测配置
backtest:
  initial_cash: 1000000
  benchmark: "000300.SH"
  evaluation:
    risk_free_rate: 0.03
    trading_days_per_year: 252

# 日志配置
logging:
  level: "INFO"
  format: "%(asctime)s - %(name)s - %(levelname)s - %(message)s"
  file: "logs/app.log"
  max_size: 10485760
  backup_count: 5
```

### 策略配置

每个策略有自己的配置文件，例如：

```yaml
# strategies/01-trend-following/config.yaml

# 策略配置
strategy:
  name: "TrendFollowing"
  description: "趋势跟踪策略"
  
  # 参数配置
  params:
    ma_short: 5
    ma_long: 20
    entry_threshold: 0.02
    exit_threshold: 0.05
    stop_loss: 0.08
    take_profit: 0.15
    
  # 交易配置
  trading:
    position_ratio: 0.3
    holding_period: 20
```

## 常见问题

### Q1: 安装依赖时出错

**问题**：`pip install -r requirements.txt` 失败

**解决方案**：

```bash
# 1. 更新 pip
pip install --upgrade pip

# 2. 使用国内镜像
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

# 3. 分步安装
pip install pandas numpy
pip install akshare
pip install matplotlib
```

### Q2: 数据获取失败

**问题**：`akshare` 或 `tushare` 数据获取失败

**解决方案**：

```bash
# 1. 检查网络连接
ping akshare.org

# 2. 检查数据源配置
vim config/default.yaml

# 3. 使用缓存数据
# 数据会自动缓存到 data/cache/
```

### Q3: 内存不足

**问题**：运行时内存不足

**解决方案**：

```bash
# 1. 减小数据量
python examples/simple_backtest.py --n-stocks 3 --periods 100

# 2. 使用更小的 batch size
# 在策略配置中调整
```

### Q4: GPU 不可用

**问题**：`torch.cuda.is_available()` 返回 False

**解决方案**：

```bash
# 1. 检查 NVIDIA 驱动
nvidia-smi

# 2. 安装 CUDA 版本的 PyTorch
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu128

# 3. 使用 CPU 版本
# 项目支持 CPU 运行，只是速度较慢
```

### Q5: 端口被占用

**问题**：Jupyter Notebook 端口被占用

**解决方案**：

```bash
# 1. 使用其他端口
jupyter notebook --port 8888

# 2. 关闭占用端口的进程
lsof -i :8888
kill -9 <PID>
```
