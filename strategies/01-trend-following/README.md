## 策略说明

**策略名称**: 趋势跟踪策略 (Trend Following)

**策略类型**: 趋势跟踪

**适合市场**: 牛市/熊市（趋势明显的市场）

**预期年化**: 20%-50%

**最大回撤**: 20%-30%

**资金需求**: 低

**难度等级**: ⭐⭐

## 策略原理

### 核心思想

顺势而为，价格沿趋势方向运动。在趋势形成时入场，在趋势反转时离场。

### 基本假设

1. 市场存在趋势（非完全随机游走）
2. 趋势一旦形成，倾向于延续
3. 在趋势结束前入场，在趋势反转前离场

### 信号生成

本策略使用双均线系统：

```
买入条件（金叉）:
1. 短期均线上穿长期均线
2. 成交量放大确认

卖出条件（死叉）:
1. 短期均线下穿长期均线
2. 或触发止损/止盈
```

### 技术指标

| 指标 | 参数 | 说明 |
|------|------|------|
| MA 短期 | 5 日 | 快速反应价格 |
| MA 长期 | 20 日 | 代表中期趋势 |
| 成交量 | - | 确认信号有效性 |

## 配置说明

### 策略参数

编辑 `config.yaml` 文件：

```yaml
strategy:
  name: "趋势跟踪策略"
  
  # 核心参数
  params:
    ma_short: 5         # 短期均线
    ma_long: 20         # 长期均线
    volume_confirm: true # 成交量确认
  
  # 交易参数
  position_size: 0.1    # 单次交易仓位（10%）
  stop_loss: 0.05       # 止损比例（5%）
  take_profit: 0.15     # 止盈比例（15%）
  
  # 回测参数
  initial_cash: 1000000  # 初始资金
  commission: 0.00025    # 佣金费率
  slippage: 0.001        # 滑点
```

### 参数说明

| 参数 | 默认值 | 说明 | 调整建议 |
|------|--------|------|---------|
| ma_short | 5 | 短期均线周期 | 减小更敏感，增大更稳定 |
| ma_long | 20 | 长期均线周期 | 减小更敏感，增大更稳定 |
| position_size | 0.1 | 单次仓位 | 根据风险承受能力调整 |
| stop_loss | 0.05 | 止损比例 | 趋势策略建议 5%-10% |
| take_profit | 0.15 | 止盈比例 | 建议大于止损的 2 倍 |

## 风险提示

⚠️ **重要风险提示**：

1. **历史业绩不代表未来** - 回测结果基于历史数据，未来表现可能不同

2. **震荡市风险** - 趋势策略在震荡市会反复止损

3. **过拟合风险** - 参数可能在历史数据上过拟合

4. **交易成本** - 实际交易成本可能高于回测假设

5. **流动性风险** - 实际交易可能面临流动性不足

### 适用人群

- ✅ 有一定量化基础的投资者
- ✅ 能够承受 20% 以上回撤
- ✅ 有充足时间监控策略运行
- ❌ 不适合风险厌恶型投资者
- ❌ 不适合期望保本保收益的投资者

### 使用建议

1. **小额试水** - 先用小资金验证策略有效性
2. **持续监控** - 定期检查策略表现
3. **及时止损** - 策略失效时及时停止
4. **分散投资** - 不要将所有资金投入单一策略
