## 策略说明

**策略名称**: 多因子选股策略 (Multi-Factor)

**策略类型**: 多因子

**适合市场**: 所有市场

**预期年化**: 15%-30%

**最大回撤**: 10%-20%

**资金需求**: 中

**难度等级**: ⭐⭐⭐⭐

---

## 策略原理

### 核心思想

从多个维度（因子）评估股票，综合评分后选择得分最高的股票。

### 因子类型

| 因子类型 | 具体因子 | 说明 |
|---------|---------|------|
| 价值因子 | PE、PB、PS | 越低越好 |
| 成长因子 | 营收增速、利润增速 | 越高越好 |
| 动量因子 | N 日收益率 | 越高越强 |
| 质量因子 | ROE、毛利率 | 越高越好 |
| 规模因子 | 市值 | 小市值效应 |

### 策略流程

```
1. 股票池筛选 → 剔除 ST、新股等
2. 因子计算 → 计算各因子值
3. 因子标准化 → Z-Score 标准化
4. 因子加权 → 综合评分
5. 排名选股 → 买入前 N 名
6. 定期调仓 → 周度/月度
```

## 配置说明

```yaml
strategy:
  name: "多因子选股策略"
  
  factors:
    value:
      weight: 0.3
      metrics: [pe, pb, ps]
    growth:
      weight: 0.3
      metrics: [revenue_growth, profit_growth]
    momentum:
      weight: 0.2
      metrics: [return_20d, return_60d]
    quality:
      weight: 0.2
      metrics: [roe, gross_margin]
  
  trading:
    holdings: 10        # 持有股票数量
    rebalance_period: 20 # 调仓周期（交易日）
```

## 风险提示

⚠️ **主要风险**:

1. **因子失效** - 因子可能随市场变化失效
2. **数据质量** - 财务数据可能不准确
3. **调仓成本** - 频繁调仓增加交易成本
