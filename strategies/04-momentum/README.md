# 动量策略

> 强者恒强，过去表现好的股票未来继续表现好

---

## 策略说明

**策略名称**: 动量策略 (Momentum)

**策略类型**: 动量

**适合市场**: 牛市

**预期年化**: 20%-50%

**最大回撤**: 20%-35%

**资金需求**: 低

**难度等级**: ⭐⭐

---

## 策略原理

### 核心思想

过去表现好的股票，未来继续表现好（中期动量效应）。

### 信号生成

```
选股条件:
1. 过去 N 日收益率排名前 N
2. 相对强度指标高
3. 突破 N 日新高

卖出条件:
1. 动量反转
2. 跌破均线
3. 止损触发
```

---

## 配置说明

```yaml
strategy:
  name: "动量策略"
  
  params:
    momentum_period: 60  # 动量周期
    top_n: 10            # 选股数量
    rebalance_period: 5  # 调仓周期
  
  trading:
    position_size: 0.1
    stop_loss: 0.08
```

---

## 风险提示

⚠️ **主要风险**:

1. **动量反转** - 强势股可能突然反转
2. **追高风险** - 可能买在高点
3. **震荡市亏损** - 震荡市反复止损
