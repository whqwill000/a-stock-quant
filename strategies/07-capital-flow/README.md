# 资金流策略

> 跟随主力资金，资金流向决定价格走向

---

## 策略说明

**策略名称**: 资金流策略 (Capital Flow)

**策略类型**: 资金流

**适合市场**: 所有市场

**预期年化**: 20%-50%

**最大回撤**: 20%-35%

**资金需求**: 低

**难度等级**: ⭐⭐

---

## 策略原理

### 核心思想

资金是推动价格上涨的直接动力，跟随主力资金流向交易。

### 资金类型

| 资金类型 | 说明 | 跟踪方式 |
|---------|------|---------|
| 北向资金 | 外资，"聪明钱" | 每日净流入 |
| 龙虎榜 | 游资动向 | 营业部席位 |
| 大宗交易 | 机构动向 | 折价/溢价 |
| 主力资金 | 大单流向 | 净流入统计 |

### 信号生成

```
买入条件:
1. 北向资金连续净流入
2. 或主力资金大幅净流入
3. 或登上龙虎榜买入榜

卖出条件:
1. 资金流向反转
2. 或触发止损
```

---

## 配置说明

```yaml
strategy:
  name: "资金流策略"
  
  params:
    net_inflow_threshold: 10000000  # 净流入阈值 1000 万
    consecutive_days: 3              # 连续流入天数
  
  trading:
    position_size: 0.1
    stop_loss: 0.05
```

---

## 风险提示

⚠️ **主要风险**:

1. **数据滞后** - 资金流数据可能滞后
2. **假流入** - 主力可能对倒出货
3. **接盘风险** - 可能成为接盘侠
